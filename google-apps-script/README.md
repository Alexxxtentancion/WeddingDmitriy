# Google Apps Script для RSVP

## Структура таблицы

Создайте лист с именем **RSVP**. Первая строка — заголовки:

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| Дата и время | Имя | Присутствие | Гостей | Еда | Напитки | Комментарий |

## Установка

1. Откройте [Google Таблицы](https://sheets.google.com) и создайте новую таблицу
2. Переименуйте первый лист в `RSVP` и добавьте заголовки из таблицы выше
3. **Расширения → Apps Script**
4. Удалите содержимое редактора и вставьте код из [`Code.gs`](Code.gs)
5. **Развернуть → Новое развёртывание → Тип: Веб-приложение**
   - Выполнять от имени: **Я**
   - У кого есть доступ: **Все**
6. Скопируйте URL развёртывания (заканчивается на `/exec`)
7. Вставьте URL в `.env` как `VITE_GOOGLE_SCRIPT_URL`

При изменении скрипта создайте **новое развёртывание** (версия обновится).
